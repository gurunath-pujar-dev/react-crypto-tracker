## ✨ Features

- Real-time data of top cryto currencies 
- Line chart for each coin for price tracking

---

## ⚙️ Execution steps:

Clone & Install dependencies:

```bash
git clone https://github.com/gurunath-pujar-dev/react-crypto-tracker.git
cd react-crypto-tracker
npm install
```


Run command:
```bash
npm run dev
```

---

##  Usage

Access the Crypto web-app at: http://localhost:5173
